# James Palma Engineering Portfolio

A React + TypeScript portfolio for research, engineering projects, embedded systems, and computational engineering work.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

This project is configured for GitHub Pages through the `gh-pages` package:

```bash
npm run deploy
```

For repository Pages settings, use the `gh-pages` branch after deployment.
